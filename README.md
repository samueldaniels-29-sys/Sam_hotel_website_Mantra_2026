# Sam's Hotel - Sample Hotel Website

This is a beginner-friendly sample frontend website for the MANTRA 2026 Summer School assignment.

## Website Type
Hotel Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Rooms
- Facilities
- Gallery
- Booking
- Contact

## Features
- Responsive design for mobile, tablet and desktop
- SEO tags on every page (title, meta description, meta keywords, alt text)
- Mobile navigation menu
- Room booking form validation using JavaScript (required fields, email, phone, check-in/check-out dates)
- Room cost calculator using JavaScript
- Gallery filter using JavaScript

## How to Run
Open `index.html` in any browser.

## Netlify Deployment Steps
1. Create a GitHub repository.
2. Upload all files from this folder.
3. Login to Netlify.
4. Click "Add new site" > "Import an existing project".
5. Connect GitHub repository.
6. Deploy the website.
